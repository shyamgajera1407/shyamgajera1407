# 🌌 Neon Float UI (Flutter)

A futuristic animated UI built with Flutter featuring neon glow effects, floating cards, and glassmorphism design.

---

## ✨ Features

* 🌈 Animated gradient background
* 💎 Glassmorphism UI
* 🔥 Neon glow effects
* 🎈 Floating card animation
* ⚡ Smooth transitions
